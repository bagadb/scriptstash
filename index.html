[ How to Use ]

> Method 1 - Manual Method
  • list -

    wget -q -O - "https://scriptstash.herokuapp.com/list"

  • run -

    wget -q -O - "https://scriptstash.herokuapp.com/s/[ script path ]" | bash -s [param1] [param2] ...

> Method 2 - One script to end 'em all!
> Download this ( link to readymade script )
  • Add executable permission:

    chmod +x scriptsta.sh

  • list -

    ./scriptsta.sh list

  • run -

    ./scriptsta.sh [script_name] [param1] [param2] ...
